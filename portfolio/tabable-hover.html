Tabable Hover Effect

See the Pen Tabable Hover for Inline Image Gallery by Travis John (@travis_john) on CodePen.

  • Platform: Codepenopens in new tab
  • Date: June 2019
  • Goal: Accessibility

I created this prototype with accessibility in mind. Hover effects are not typically accessibile to users with disabilities, so I wrote a function to make the effect accessibile on hover and on focus. In addition I wrote a polyfill so that the hover effect is disabled on mobile devices so that disabled users can still access the conent on their phones.

This prototype is also viewable on this website's home page.
